Usage:

  1) sbt assembly
  
  2) ./start.sh                   - Start ground-control and one drone
  
  3) ./fake_drone                 - Start 3 drone emulators
  
  4) ./terminal.sh                - And one terminal to rule them all (type 'help' in terminal to know how)
  
  5) Go to http://localhost:9000/ - WEB UI 
  
  6) ./stop.sh                    - Stop all
